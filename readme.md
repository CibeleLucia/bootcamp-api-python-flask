# Bootcamp API - Usando Python com Flask

```bash
# Clonar repositório
git clone https://github.com/vinilima2/bootcamp-api-python-flask
```

```bash
# Acessar projeto
cd bootcamp-api-python-flask
```

```bash
# Instalar dependências
pip install -r requirements.txt
```

```bash
# Executar projeto
flask --app main run
```
